L’analyse du sentiment (ou de la tonalité), aussi appelé opinion mining, est une notion beaucoup évoquée mais souvent mal comprise. Elle est un processus qui permet de déterminer la tonalité émotionnelle qui se cache derrière une série de mots. Cette analyse est utilisée pour mieux comprendre la perception, les opinions et les émotions exprimées dans une mention en ligne.Dans cette application la méthode d'analyse est appliquée sur les données Twitter, aprés leurs avoir collecter à l'aide de l'API de recherche twitter et les stocker dans la base de données neo4j et les traiter à l'aide des méthodes de traitement de langages naturels.

Céleri est une file d'attente de tâches avec un accent sur le traitement en temps réel, tout en prenant en charge la planification des tâches. Redis est un courtier de messages. Cela signifie qu'il gère la file d'attente des "messages" entre Django et Céleri. Django est un framework web conçu pour les perfectionnistes avec des délais.Tous les trois travaillent ensemble pour faire de la magie en temps réel.
